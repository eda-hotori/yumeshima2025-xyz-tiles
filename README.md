025年大阪・関西万博会場　3D都市モデル（Project PLATEAU）で配布されている航空写真（GeoTIFF形式）をxyzタイル形式に変換したものです。

利用規約は配布元ページに準じます。
https://www.geospatial.jp/ckan/dataset/plateau-27999-osaka-shi-2025
> どなたでも無償で利用できますが、商業目的、販売促進、広告利用、商品化などの営利目的での使用は一切認められていません。

PLATEAU SDK for Unity( https://project-plateau.github.io/PLATEAU-SDK-for-Unity/ )のインポートツールで使用する場合、「土地起伏」->「航空写真または地図を貼り付ける」のURLを以下に設定してください。

```https://eda-hotori.github.io/yumeshima2025-xyz-tiles/{z}/{x}/{y}.jpg```
